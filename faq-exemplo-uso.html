Exemplo de Uso do FAQ - Roxy Games

Como usar as funções do FAQ:

1. Adicionar uma nova pergunta:

FAQ.adicionarPergunta("Nova pergunta?", "Nova resposta.");

2. Remover uma pergunta (por índice):

FAQ.removerPergunta(0); // Remove a primeira pergunta

3. Atualizar todas as perguntas:

const novoFAQ = [ { question: "Pergunta 1", answer: "Resposta 1" }, { question: "Pergunta 2", answer: "Resposta 2" } ]; FAQ.atualizarFAQ(novoFAQ);

4. Acessar os dados do FAQ:

console.log(FAQ.faqData); // Ver todas as perguntas no console

5. Recriar o FAQ:

FAQ.inserirFAQ(); // Recria o FAQ na página

Estrutura dos Dados:

O FAQ usa um array de objetos com a seguinte estrutura:

const faqData = [ { question: "Sua pergunta aqui?", answer: "Sua resposta aqui." } ];

Personalização:

Para modificar as perguntas, edite o array faqData no arquivo faq.js.

Para alterar o estilo, modifique as classes CSS no arquivo style.css.
